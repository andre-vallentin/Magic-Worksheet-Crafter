Erstelle Quellenangaben immer nach dieser Vorlage:

- einem Kommentar `<!-- References -->`
- einen Header2 `## Quellen`

# Formatierung:
- Informationen aus meinem Trainingswissen: `[Trainingswissen] Kurze Beschreibung des Inhalts → Aufgabe(n): #, #`
- Informationen aus einer Websuche: `[Websuche] <URL>, Datum von heute → Aufgabe(n): #, #`
- Informationen aus PDFs: `[PDF-Name] S. #(–#)[, Kapitel: "Überschrift"] → Aufgabe(n): #, #`
- Seitennummer ist Pflicht; Kapitelüberschrift angeben sofern vorhanden (Erfine keine Kapitelüberschriften oder Kontexte selbst)
- Informationen aus Bildern: `[Bild-Name] → Aufgabe(n): #, #`

# Beispiel
```
<!-- Section -->
<!-- References -->
## Quellen

[Nährstoffe.pdf] S. 336–337 → Aufgaben: 1, 2, 3

[Energie-für-dich.png] → Aufgaben: 1, 2
```
