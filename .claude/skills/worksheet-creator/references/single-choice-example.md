Erstelle SingleChoice bzw. 'Wahr oder Falsch'-Aufgaben immer nach dieser Vorlage:

- einem Kommentar `<!-- SingleChoice -->`
- einen Header2 `## Aufgabentitel`
- eine Aufgabenbeschreibung
- eine Tabelle mit drei Spalten mit "Aussage, Richtig, Falsch" 
- im Lösungsabschnitt trage in die jeweilige Spalte ein "x" ein.

# Beispiel

## Aufgabenstellung

```
<!-- SingleChoice -->
### Aufgabe 5: Wahr oder Falsch?
Entscheide, ob die folgenden Aussagen richtig oder falsch sind.

| Aussage | Richtig | Falsch |
|---|---|---|
| Testosteron wird im Gehirn gebildet. | | |
| Der Stimmbruch tritt zwischen 11 und 16 Jahren auf. | | |
| Jungen wachsen in der Pubertät besonders langsam. | | |
```
